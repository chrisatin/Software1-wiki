Wiki de Ingeniería de Software I

Fundamentos, modelos y procesos del desarrollo de software

Propósito de la Asignatura

El objetivo de este curso es:

  • Fomentar el significado de la ingeniería en el proceso de desarrollo de software
  • Aplicar procesos industriales utilizando las mejores prácticas de producción internacional
  • Seleccionar, con justificación, el modelo de ciclo de vida apropiado para el diseño, desarrollo y mantenimiento de productos de software
  • Conceptuar, entender y aplicar métodos de análisis y especificación de requisitos en un sistema de software pequeño

1. Ciclo de Vida del Software

Modelos de ciclo de vida del proceso software y el software como producto y sus métricas.

2. Requisitos

Elicitación y especificación de requisitos, técnicas de análisis y prototipado.

3. Arquitectura de Software

Fundamentos, tipos y patrones de diseño, diseño UML (Casos de Uso, Análisis, Diseño, Despliegue).

4. Diseño Detallado

Modelos de implementación y pruebas, selección de herramientas y métricas.

5. Pruebas

Planificación y principios, técnicas de caja negra y caja blanca, inspecciones y revisiones.

6. Operación y Mantenimiento

Procesos, reingeniería y reutilización.
